Say Hello to coding

發表於 2024-04-25 分類於 learningPlan
文章字數： 750 所需閱讀時間 ≈ 1 分鐘

Foreword

數算起來，我接觸到 Coding 的時間算早。

大學某一年的暑假，為了想要多瞭解 Coding，主動跟國中同學借 Borland C++ Builder 的相關書籍來讀，我記得，借來的時候，被上下 2 冊厚重的書本給嚇了一大跳。然而，暑假已經走完，那 2 本書幾乎沒有被我動過，就原封不動地還給同學。

那時候，一方面，我自己懵懂未知；一方面，則是被厚重的書本與一堆看不懂的詞震驚到，從此，Coding 這件事，就一直被我給擱著不動。

也可以說，我是被嚇懵了，不清楚可以從何處尋求學習資源，並進一步突破。

Get in touch

僅管如此，這件事並沒有被我遺忘，而是深埋在心裡。

直到近年，因緣際會接觸到網路的教學資源，這些資源有文字的，也有以影片的方式呈現的。

起初，我想學的是 Swift，在網路上找到一些資源後，開始學習才發現，當年僅僅只是「接觸」Borland C++ Builder 相關書籍的我，壓根沒有吸收到任何跟 Coding 有關的知識養份，充其量只能說當時候「已播下了一顆想要學習的種子」。

只可惜，這一波學習，我並沒有維持很久，也因此，我也只學到非常皮毛的東西。

但是，也在想要學習 Swift 的過程，再次碰觸 C++。

只不過呀！只不過！

要我現在講一點跟 C++ 有關的，我最多只能抱著不確定的心情，講出個 cin、cout，沒有持續學習的我，根本不敢說自己認識 C++。

Restart

設立這支部落格的目的，主要是我想放掉以前與 C++ 相處的種種，讓自己歸零，重新開始。

希望自己能夠用比較輕鬆愉快的心情面對 Coding，從而慢慢累積相關的知識。

至於，要學習到什麼程度？

就目前而言，我先把目標設定在「消化完目前手邊的教學資源」。

走完這一段之後，再評估我學到什麼程度，其餘再說。
